## Prompt

Explain this regular expression: {{regex}}

For each significant part (anchors, quantifiers, groups, character classes), describe what it does and give a short example of text it would match.

## Usage

Replace `{{regex}}` with any regex (escape backslashes in JSON if needed). Use when debugging or learning an unfamiliar pattern.
